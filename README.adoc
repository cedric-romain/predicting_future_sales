[source,sh]
----

==== Issues with `XGBoost`

===== Issue: *XGBoost Library (libxgboost.dylib) could not be loaded.*

[source,sh]
----
conda install -c conda-forge py-xgboost
----


===== Issue: *cannot import name 'CUDF_concat' from 'xgboost.compat'*

[source,sh]
----
brew install xgboost
----
